The map component now includes:

Interactive AI Insights Panel

AI insights now feature:

  • Interactive charts for deeper data exploration

Data Source Toggle

Users can now switch between:

  • Simulated data for testing and demos

  • Visual indicators showing current data source
  • Confirmation dialog for switching to live data

TBWA Theme Update

Updated styling includes:

  • New color palette with Azure Blue (#0052CC) and Yellow-Orange (#FFAB00)
  • Consistent application of theme colors throughout the UI
  • Enhanced visual feedback for interactive elements
  • Theme switcher with multiple options
