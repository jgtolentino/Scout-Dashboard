Project Scout Dashboard

Analysis Overview Component

This component provides high-level metrics about the quality, volume, and impact of brand mentions analysis, along with a detailed breakdown of key performance indicators.

Integration Instructions
How to Add to Project Scout Dashboard
  1. Include the stylesheet: 
    <link rel="stylesheet" href="styles/analysis-overview.css">
  2. Include the JavaScript component: 
    <script src="components/analysis-overview.js"></script>
  3. Add a container div in your dashboard: 
    <div id="analysisOverviewContainer"></div>
  4. Initialize the component after DOM is loaded: 
    const analysisOverview = new AnalysisOverview({
  container: '#analysisOverviewContainer',
  apiEndpoint: '/api/analysis',
  refreshInterval: 60000 // 1 minute
});
Recommended Placement

The Analysis Overview component is designed to be placed at the top of the dashboard, after the filter bar and before the main charts. Ideal placement would be immediately before the "Brand Mentions Over Time" chart in the main dashboard.
